# MoveNet Sample

![demo](demo.gif)

This is an example project which integrates the NatML deep learning API with the NatDevice media device API for realtime pose detection.

## Requirements
- Unity 2020.3+

## Supported Platforms
- Android API level 24+
- iOS 13+
- macOS 10.15+
- Windows 10 (64-bit)
